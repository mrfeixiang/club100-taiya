# Emonda 胎压顾问 · Tire Pressure Advisor

输入体重,推荐 Trek Emonda(及任意公路车)的前后轮合适胎压。基于经典的 **15% 轮胎落差法(Frank Berto tire-drop)**,给出一个有依据的起始参考值。

**在线使用:** https://mrfeixiang.github.io/club100-taiya

> 单文件、纯前端、无依赖、离线可用。把 `index.html` 双击即可本地打开。

---

## 用法

拖动体重滑块,实时给出前后轮胎压(psi 与 bar)。可调整:

- **轮胎宽度** 23 / 25 / 28 / 30 / 32c(Emonda 现款官方上限约 28mm)
- **内胎类型** 开口胎(有内胎)/ 真空胎 Tubeless
- **路面状况** 光滑 / 一般 / 粗糙
- **进阶:** 整车 + 装备重量(默认 8.5kg)

## 计算方法

核心是让轮胎在负重下压缩约 **15% 直径**,此时滚阻与舒适度接近最优。公式(标定自 Berto 图表):

```
胎压(psi) ≈ C(胎宽) × √(每轮负重kg) × 路面系数 × 内胎系数
```

- 系统重量 = 体重 + 整车装备,按 **前 45 / 后 55** 分配到两轮(后轮负重更大 → 胎压更高)
- 胎宽越宽,同重量所需气压越低,按 `width^-1.7` 缩放
- 真空胎 −10%;路面越粗糙,气压越低

关键参数都集中在 `index.html` 底部的 `<script>` 里,想校准直接改:
`C25`(整体高低)、`WIDTH_EXP`(胎宽敏感度)、`FRONT`/`REAR`(前后配重)。

## 免责声明

这是**起始参考值**,不是定论。请从推荐值出发、每次 ±3psi 用手感微调。
**切勿超过轮胎侧壁与轮圈标注的额定气压范围。** 真空胎过低会脱圈漏气,有内胎过低会蛇咬破胎。
